# consumer-python-uv fixture

Canonical Python/uv consumer project for substrate's hardening-program
validation matrix (H2.1). Used by the stub-agent pipeline e2e (H2.2) and the
nightly live smoke (H2.3).

- `bootstrap.sh` creates the uv env and proves the base suite green.
- Story 1-1 is cleanly implementable; story 1-2 is the verification-trip
  vehicle for stub scenarios.
- `uv.lock` is intentionally gitignored here (regenerated by bootstrap) to
  keep the fixture platform-neutral; `substrate init` still detects uv via
  the lock generated at bootstrap time.
