# CORE Workflows

## Available Workflows in core

**brainstorming**
- Path: `_bmad/core/workflows/brainstorming/workflow.md`
- Facilitate interactive brainstorming sessions using diverse creative techniques and ideation methods. Use when the user says "help me brainstorm" or "help me ideate".

**party-mode**
- Path: `_bmad/core/workflows/party-mode/workflow.md`
- Orchestrates group discussions between all installed BMAD agents, enabling natural multi-agent conversations. Use when user requests "party mode" only.


## Execution

2. READ its entire contents and follow its directions exactly
3. Save outputs after EACH section
